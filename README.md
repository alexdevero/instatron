# Instagram desktop uploader

A simple Instagram desktop uploader & client app build with electron. Share your photos and images just like on your mobile phone!

### Table of contents

* [Install](#install)
* [Usage](#usage)
* [Code of Conduct](#code-of-conduct)
* [License](#license)

### Install

#### Clone this repo

```
git clone https://github.com/alexdevero/electron-react-webpack-boilerplate.git
```

#### Install dependencies

```
npm install
```
or
```
yarn
```

### Usage

#### Run the app

```
npm run start
```
or
```
yarn run start
```

#### Build the app

```
npm run package
```
or
```
yarn run package
```
