# Instatron - Instagram desktop uploader

A simple Instagram desktop uploader & client app build with electron. Share your photos and images just like on your mobile phone!

### Features

- Upload & post images
- Browse photos
- Follow/unfollow users
- Bookmark photos
- Like photos
- Manage your profile
- Manage your settings
- Watch moments

### Install

#### Clone this repo

```
git clone https://github.com/alexdevero/instatron.git
```

#### Install dependencies

```
npm install
```
or
```
yarn
```

### Usage

#### Run the app

```
npm run start
```
or
```
yarn run start
```

#### Build the app

```
npm run package
```
or
```
yarn run package
```
